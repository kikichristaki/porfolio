featured project one

3D Line Message

HTML5 / WebGL / JS

The 3D Line Message is a web application that allows users to write a message on the screen and then the message is displayed using colorfoul lines and 3D effects. The project focuses on the idea of volume rendering using lines to create aesthetically pleasing messages over the web. It was an experiment on 3D graphics in web browsers made with WebGl technology and utilising three.js javascript library.


Technologies used: WebGl, Javascript/Three.js

featured project two

Visualisation in 3D spaces

Unity / Data Visualisation

This project is conducted in partnership with the company "720 Degrees". The purpose of the project is to investigate the possibilities of utilising 3D models of buildings & indoor spaces for information and data visualisation as well as navigation of the customers.

Interesting fact is that we are using Unity3D, a game engine, instead of a data visualisation framework as the possibilities that Unity offers for 3D visualisation are more robust. The drawback is that connecting a Unity project with a rest API, for accessing the visualised data, is quite more complex.


Technologies used: Unity3D, WebGL, Javascript/Json, Ruby on rails, restAPI

featured project three

My expressive Avatar

Human-Computer Interaction experiment

My Expressive Avatar is part of a large-scale project on the field of Expression-based Interaction. It is a collaboration between KTH and The University of Maastricht that aims to enhance communication by affective content through facial expression recognition. It involves the mapping of real facial expressions onto virtual avatars.

It consists by a virtual chat room where the desired avatar is displayed. The avatar is able to change facial expressions according to the expression of the remote user. The progect currently is at the stage of experimenting with real users to study actual user engagement.


Technologies used: Unity3D, Faceshift

featured project six

Interior Ninja

Virtual Reality application

Interior Ninja is a virtual reality app for interior designing. It has been developed during AEC Hackathlon in Helsinki and the porpuse was to propose a new way to improve customer's experience during the choosing and buying process.

The app is designed for a bathroom company. The customer, wearing Oculus rift, can navigate inside a virtual model of his/her bathroom and change the tiles in the surfaces by choosing from tile's samples provided. The app automatically calculates the final price based on the tiles cost, taken by the company's catalog, and the surface to be covered.


Technologies used: Unity3D, Maya, Blender, Oculus Rift
